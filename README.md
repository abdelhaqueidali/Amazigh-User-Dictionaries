# Amazigh-User-Dictionaries
Here you can find user dictionaries for text prediction on keyboards. These dictionaries are created from ircam datasets of DGLAi dictionary and Amsfti for verbs.
</br>
Amazigh.txt is word per line file.
</br>
Use UDM app for importing it to the right language code.Regarding Tifinagh keyboards, for Samsung keyboard it is ber, for Gboard it is ber-Tfng, and tf is what's used on Multiling O Keyboard. check out your keyboard what language code it is using for Amazigh.
</br> It is usually: ber, ber-Tfng, zgh, zgh-Tfng
for the latin version, usually kab for kabyle, and on Gboard it is ber-Latn or ber.
